# Fluxo

Fluxo é uma aplicação de gerenciamento financeiro pessoal desenvolvida para auxiliar no controle de receitas, despesas, categorias e metas financeiras.

O projeto está sendo desenvolvido utilizando Java e Spring Boot como backend, seguindo boas práticas de arquitetura em camadas (Controller, Service e Repository) e persistência de dados com Spring Data JPA e PostgreSQL.

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Funcionalidades (em desenvolvimento)

- Cadastro de receitas e despesas
- Gerenciamento de categorias
- Gerenciamento de metas financeiras
- Dashboard financeiro
- Relatórios e exportação de dados
